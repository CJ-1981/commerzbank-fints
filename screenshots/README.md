# Screenshots Directory

Place your application screenshots in this directory and update the main README.md to reference them.

## How to Add Screenshots

1. **Run the application**: `python commerzbank_fints_qt_desktop_app.py`

2. **Take a screenshot**: 
   - **macOS**: Press `Cmd + Shift + 4` and select the window
   - **Windows**: Press `Win + Shift + S` and select the area
   - **Linux**: Use your distribution's screenshot tool

3. **Save the screenshot**: Save as `app-screenshot.png` in this directory

4. **Update README.md**: Replace the placeholder line in the main README.md with:
   ```markdown
   ![Application Screenshot](screenshots/app-screenshot.png)
   ```

## Screenshot Guidelines

- Show the main application window with sample data
- Highlight key features like the batch table, validation indicators, and terminal log
- Use a resolution between 1200x800 and 1920x1080 pixels
- Ensure sensitive information (account numbers, real names) is obscured
- PNG format recommended for quality

## Alternative Screenshot Names

You can also use descriptive names:
- `main-interface.png` - Primary application window
- `batch-management.png` - Table with multiple payout rows
- `validation-demo.png` - IBAN validation in action
- `photoTAN-auth.png` - Authentication dialog
